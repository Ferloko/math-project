# Proyecto Web Minimalista

Un sitio web moderno, minimalista y responsivo diseñado para presentar un proyecto con información, cronología, galería de fotos y formulario de contacto.

## 🎨 Características

- **Diseño Minimalista**: Interfaz limpia y moderna con enfoque en el contenido
- **Totalmente Responsivo**: Funciona perfectamente en móviles, tablets y escritorio
- **Paleta de Colores Definida**: Esquema de colores coherente y profesional
- **Tipografía Profesional**: Fuentes Inter y Playfair Display para legibilidad y elegancia
- **Animaciones Sutiles**: Transiciones suaves y micro-interacciones
- **Galería Interactiva**: Sistema de filtrado y lightbox para imágenes
- **Cronología Visual**: Línea de tiempo interactiva del proyecto
- **Formulario de Contacto**: Validación en tiempo real y notificaciones
- **Navegación Intuitiva**: Menú sticky con scrollspy y menú móvil

## 🚀 Tecnologías Utilizadas

- **HTML5**: Semántica moderna y accesibilidad
- **CSS3**: Flexbox, Grid, animaciones y variables CSS
- **JavaScript Vanilla**: Sin dependencias, código optimizado
- **Font Awesome**: Iconos profesionales
- **Google Fonts**: Tipografía de alta calidad

## 🚀 Cómo Usar

1. **Clonar o descargar** los archivos del proyecto
2. **Abrir `index.html`** en tu navegador preferido
3. **Navegar** por las diferentes secciones
4. **Personalizar** el contenido según tus necesidades

## 🛠️ Personalización

### Cambiar Colores
Edita las variables CSS en `:root` al inicio de `styles.css`:

```css
:root {
    --color-primary: #tu-color;
    --color-secondary: #tu-color;
    /* ... otras variables */
}
```

### Modificar Contenido
Edita directamente el HTML en `index.html`:
- Cambia textos y títulos
- Reemplaza imágenes (usa URLs o archivos locales)
- Ajusta información de contacto
- Modifica la cronología

### Agregar Nuevas Imágenes
1. Reemplaza las URLs en el HTML
2. O usa archivos locales en la misma carpeta
3. Asegura consistencia en tamaños (400x300px recomendado)
